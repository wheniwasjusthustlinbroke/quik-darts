## How to Play

1. Select number of players (1-4)
2. Enter player names
3. Adjust skill level (affects accuracy)
4. Click **Start Game**

### Throwing:
- **Move mouse** to aim at the dartboard
- **Hold click** to charge the power meter
- **Release** when the power bar is in the green "perfect" zone for best accuracy
- Throw 3 darts per turn, then it's the next player's turn

### Scoring:
- Hit exactly 0 to win
- Must finish on a **double** (outer ring or bullseye)
- Going below 0 or leaving 1 = **BUST** (turn ends, score resets)
